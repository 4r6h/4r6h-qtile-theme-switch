# Requeriments

```bash
sudo pacman -S python2 python3 python-pip qtile
```

# Clone the config:

```bash
git clone https://github.com/antoniosarosi/dotfiles
cp -r dotfiles/.config/qtile .config/qtile
```

# Usage:

```bash
cd qtile-theme
python3 main.py
```
Coose a theme
Restart Qtile with Ctrl + Mod + R
